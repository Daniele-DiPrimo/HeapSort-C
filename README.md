# **HEAP-SORT IN C**

Il codice sorgente contiene l'algoritmo di ordinamento Heap-Sort.
Una volta eseguito il programma è possibile: 
- Scegliere la dimensione dell'array che si vuole ordinare
- Scegliere se riempire l'array manualmente o con valori casuali
  - Il range di valori casuali è di 0 - 1000 --> Facilmente modificabile ai fini dell'analisi dell'algoritmo --> Basti impostare il nuovo tetto massimo (+1) nella riga 103 --> 'arr[i] = rand() % 1001;'  
Effettuata la scelta l'algoritmo procede nell'ordinare l'array. A schermo si visualizza: 
- L'array di partenza
- L'array subito dopo l'operazione di BUILD-MAX-HEAP
- L'array finale ordinato
